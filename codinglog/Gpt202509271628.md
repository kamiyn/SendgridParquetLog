# Prompt
- HomeSearchConditions コンポーネントのコードビハインドを .razor に統合する依頼。
- ParquetController に ETag 対応とメタデータキャッシュ(60秒以内はS3問い合わせ省略)の実装依頼。

# Processing
- `SendgridParquetViewer/Components/Pages/HomeSearchConditions.razor` にパラメーターとイベント処理を内包する `@code` ブロックを追加し、コードビハインドを削除。
- `SendgridParquetViewer/Controllers/ParquetController.cs` に IMemoryCache を注入し、S3メタデータ取得を60秒キャッシュする `GetMetadataWithCacheAsync` を追加。If-None-Match ヘッダーと一致する ETag の場合は 304 を返し、S3 へのデータ取得を回避。
- `SendgridParquetViewer/Program.cs` で `AddMemoryCache` を登録し、コントローラーでのキャッシュ使用を有効化。
- `dotnet format --no-restore` を試行したが、NamedPipe Permission denied により失敗。
